## 📚 Architectures Covered

### 🟢 LeNet

📄 http://yann.lecun.com/exdb/publis/pdf/lecun-98.pdf
**Learn:** CNN basics, pooling intuition, MNIST pipeline

### 🔵 AlexNet

📄 https://papers.nips.cc/paper/4824-imagenet-classification-with-deep-convolutional-neural-networks.pdf
**Learn:** ReLU breakthrough, dropout, GPU training

### 🟣 VGG16 / VGG19

📄 https://arxiv.org/pdf/1409.1556.pdf
**Learn:** Depth scaling, feature hierarchy, parameter trade-offs

### 🟡 GoogLeNet / Inception

📄 https://arxiv.org/pdf/1409.4842.pdf
**Learn:** Multi-scale learning, inception modules, efficiency

### 🔴 DenseNet

📄 https://arxiv.org/pdf/1608.06993.pdf
**Learn:** Dense connectivity, feature reuse, gradient flow

### 🟠 MobileNet

📄 https://arxiv.org/pdf/1704.04861.pdf
**Learn:** Depthwise separable conv, edge deployment

### 🟢 EfficientNet

📄 https://arxiv.org/pdf/1905.11946.pdf
**Learn:** Compound scaling (width-depth-resolution)

### ⚫ ConvNeXt

📄 https://arxiv.org/pdf/2201.03545.pdf
**Learn:** Modern ConvNet design inspired by Transformers

### 🧠 Vision Transformer (ViT)

📄 https://arxiv.org/pdf/2010.11929.pdf
**Learn:** Patch embeddings, global self-attention

### 🌪️ Swin Transformer

📄 https://arxiv.org/pdf/2103.14030.pdf
**Learn:** Hierarchical attention, shifted windows
